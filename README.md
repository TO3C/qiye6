# 流云智炬科技官网 (qiye6)

[![GitHub Pages](https://img.shields.io/badge/GitHub%20Pages-Live-brightgreen)](https://TO3C.github.io/qiye6)

## 项目简介

流云智炬科技官方网站 - **热带生态与现代商务风**设计，专注于智慧环境监测、绿色能源、生态农业、碳中和管理等生态科技领域。

## 设计风格

**热带生态与现代商务风 (Eco-Business & Modern)**

- **生态绿**: #059669 (主色，象征环保与可持续)
- **科技蓝**: #0EA5E9 (辅助色，代表创新与科技)
- **活力橙**: #F97316 (强调色，体现活力与热情)
- **纯净白**: #F8FAFC (背景色，传达透明与专业)
- **玻璃拟态**: Glassmorphism 卡片设计
- **模块化网格**: Grid-based 清晰布局

## 核心特性

### 双语支持
- ✅ 中英文无缝切换
- ✅ 自动记忆用户选择
- ✅ 平滑过渡动画

### 视觉效果
- **玻璃拟态卡片** - 半透明毛玻璃效果
- **渐变强调** - 生态绿到科技蓝渐变
- **动态计数器** - 数据展示动画
- **滚动触发动画** - GSAP 实现流畅过渡
- **响应式设计** - 完美适配桌面/平板/手机

### 页面结构
1. **Hero 区域** - 品牌展示 + 核心数据
2. **核心技术** - 4大生态科技领域
3. **解决方案** - 4大行业定制方案
4. **新闻动态** - 最新资讯与政策解读 ✅
5. **成功案例** - 项目展示 + 合作伙伴
6. **联系我们** - 双语联系表单
7. **页脚** - 社交媒体链接

## 技术栈

- **HTML5** - 语义化结构
- **Tailwind CSS** - 原子化 CSS 框架
- **JavaScript** - 交互逻辑 + 双语切换
- **GSAP** - 高级动画库
- **Lucide Icons** - 图标系统

## 本地开发

```bash
# 进入项目目录
cd qiye6

# 启动本地服务器
python3 -m http.server 8888

# 在浏览器中打开
open http://localhost:8888
```

## 双语切换

点击导航栏右上角的 **中/EN** 按钮即可切换语言：
- **中文**: 默认语言
- **英文**: 完整英文翻译

切换后自动保存偏好设置。

## 部署

本项目使用 GitHub Pages 自动部署：

1. 推送代码到 `main` 分支
2. GitHub Pages 自动构建
3. 访问 `https://TO3C.github.io/qiye6`
